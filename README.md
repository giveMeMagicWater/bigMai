# 大麦网自动化抢票工具

## 项目概述
基于Node.js + Puppeteer的自动化抢票工具，模拟真实用户行为，实现大麦网抢票流程的完全自动化。

## 核心特性
- 🚀 **智能抢票**：精确时间控制，毫秒级响应
- 🤖 **反检测**：模拟真实用户行为，避免被识别为机器人
- 📱 **多端支持**：支持PC端和移动端抢票
- 🔄 **智能重试**：自动处理"前方拥挤"等异常情况
- 📊 **实时监控**：抢票过程实时状态显示
- ⚡ **高并发**：支持多账号同时抢票

## 技术架构
```
src/
├── core/           # 核心抢票逻辑
├── browser/        # 浏览器管理
├── strategies/     # 抢票策略
├── utils/          # 工具函数
├── config/         # 配置文件
└── index.js        # 主入口
```

## 抢票策略
1. **时间同步策略**：NTP时间同步，确保开票时间精确
2. **并发策略**：多线程并发请求，提高成功率
3. **重试策略**：智能重试机制，处理网络异常
4. **反检测策略**：随机延迟、模拟鼠标移动等

## 使用方法
1. 安装依赖：`npm install`
2. 配置参数：编辑 `config/config.js`
3. 运行程序：`npm start`
